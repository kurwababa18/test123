COMMON ISSUES & SOLUTIONS
-------------------------

Issue: "Python not found"
Solution:
  - Install Python 3.10+ from python.org
  - During install, check "Add Python to PATH"
  - Restart command prompt after installing

Issue: "Module not found" errors
Solution:
  - Make sure __init__.py files exist in core\ and ui\
  - Both files can be empty, but MUST be present
  - Check spelling is exactly: __init__.py (two underscores)

Issue: run.bat doesn't start
Solution:
  - Right-click run.bat > Edit
  - Verify it contains the batch script code
  - Try running from command prompt to see errors:
    cd C:\Users\turner\Downloads\claude111
    run.bat

Issue: Dependencies won't install
Solution:
  - Delete venv\ folder completely
  - Run run.bat again
  - Or install manually:
    python -m venv venv
    venv\Scripts\activate
    pip install -r requirements.txt

Issue: No data appearing
Solution:
  - Wait 30 seconds for initial cache
  - Check logs\polyterm.log for errors
  - Press 'R' to force refresh
  - Check internet connection
